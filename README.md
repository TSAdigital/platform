# TSA<sup>Platform</sup>

Стартовый шаблон разработан на основе [Yii 2 Basic Project Template](https://github.com/yiisoft/yii2-app-basic) и [AdminKit](https://github.com/adminkit/adminkit).

Данное приложение предоставляется без гарантий безопасности. Использование осуществляется на ваш страх и риск. Оно предназначено исключительно для закрытого (тестового) окружения.

## Рекомендации

Рекомендуется использовать PHP версии 7.4.

## Установка

### Установка через Composer

В корневом каталоге проекта выполните команду:

```bash
composer install
```

Обновите значение параметра `cookieValidationKey` в файле `config/web.php`, установив случайную строку:

```php
'request' => [
    'cookieValidationKey' => '<секретная случайная строка>',
],
```

### Настройка подключения к базе данных

Измените файл `config/db.php`, добавив актуальные данные для подключения к базе данных:

```php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=yii2',
    'username' => 'root',
    'password' => '12345678',
    'charset' => 'utf8',
];
```

### Выполнение миграций

После настройки подключения выполните команды для применения миграций:

```bash
php yii migrate
```

Это создаст необходимые таблицы и структуры данных в базе данных.

### Установка через Docker

Находясь в корне проекта, где расположен файл `docker-compose.yml`, выполните следующую команду:

```bash
docker-compose up -d
```

Эта команда создаст и запустит контейнеры, описанные в `docker-compose.yml`.

Имейте в виду, что после развертывания контейнеров может потребоваться некоторое время для завершения миграций и установки зависимостей, что может занять до 5 минут в зависимости от конфигурации окружения. Пожалуйста, проявите терпение, если контейнер не отвечает сразу — дождитесь выполнения всех команд перед попыткой доступа к проекту.

Обновите значение параметра `cookieValidationKey` в файле `config/web.php`, установив случайную строку:

```php
'request' => [
    'cookieValidationKey' => '<секретная случайная строка>',
],
```
## Тестирование

Приложение готово к эксплуатации. Вы можете открыть браузер и перейти по адресу `http://localhost:8000`, чтобы получить доступ к странице входа в личный кабинет.

Для входа используйте следующие учетные данные: **Имя пользователя:** `admin` **Пароль:** `12345678`.
